# 실험 로그 (장표용)

> 모든 점수는 층화 5-fold OOF Balanced Accuracy (제출 지표와 동일). LB는 제출한 것만.
> 규칙: CV로만 채택 판단 (public LB는 test 20%라 참고용). 누수 금지 (TE·보정은 fold 내부/OOF에서만).

## 배경 조사 (개선 방향의 근거)

- **라벨 생성 규칙 역공학** (커뮤니티, 원본 데이터 기준 정확도 100%):
  - 수면<6h + 스트레스 high → unhealthy / 수면≥7h + 스트레스 low + active → fit / 나머지 → at-risk
  - → 대회 데이터 = 규칙 + 합성 노이즈 + 결측. 점수 천장 ~0.952의 정체
- **검증된 돌파 기법** (커뮤니티 9-노트북 트레일): 수치형 exact-value TE + HistGB = +0.0009 (LB 0.95036), CV↔LB 갭 없음
- **죽은 길 확인** (동일 출처): 임계값 튜닝(-0.0001), 4-way 앙상블(-0.0002), OvR 분해(+0.0001) → 전부 노이즈 수준
- **리더보드 지형**: 1위 0.95182 / 대형 클러스터 0.95114 (동일 공개 노트북 포크 추정)
